# CallCenter 中台联调测试脚本与模板

本文档提供了用于在联调测试环境中验证人员同步与工单推送（含 UPSERT 状态流转）的完整 `curl` 测试脚本。你可以直接在终端中运行以下脚本进行测试验证。

---

## 1. 人员同步接口测试（全字段必填验证）

此脚本用于验证全字段强校验逻辑以及 `isActive` 状态控制。

```bash
curl -v -k -X POST https://101.251.208.178/api/ext/users/sync \
-H "Content-Type: application/json" \
-H "Authorization: Bearer 1c43f8a0ea1b72b4511d42c584e977e5cea21ba0cce67fcea7a65c079954011d" \
-d '[
  {
    "employeeId": "TEST_001",
    "realName": "测试员工A",
    "email": "testA@trustfar.cn",
    "phone": "13800138001",
    "department": "技术支持部",
    "position": "一线客服",
    "isActive": 1
  }
]'
```

**预期返回：**
`"code": 0, "message": "同步成功"`

---

## 2. 工单推送测试一：首次强制状态验证

此脚本用于验证首次推送新工单（无相同 `serviceNo` 时），系统是否能**强制过滤**掉 Payload 中的 `"status": "pending"`，并在系统内将其自动置为 `in_progress`（服务中）。

```bash
curl -v -k -X POST https://101.251.208.178/api/ext/tickets \
-H "Content-Type: application/json" \
-H "Authorization: Bearer 1c43f8a0ea1b72b4511d42c584e977e5cea21ba0cce67fcea7a65c079954011d" \
-d '{
  "title": "生产环境联调测试工单",
  "description": "这是一条用于测试联调的工单，验证首次强制 in_progress 状态。",
  "serviceNo": "TEST-CURL-001",
  "customerName": "VIP大客户",
  "creatorEmployeeId": "TEST_001",
  "assigneeEmployeeId": "TEST_001",
  "type": "software",
  "category1": "远程测试",
  "status": "pending"
}'
```

**预期返回：**
返回结果中，`"updated": false`，且 `"status": "in_progress"`。

---

## 3. 工单推送测试二：二次推送状态同步（秒关单）

在执行完“测试一”后，立即执行本脚本，使用**完全相同的** `serviceNo`，但将状态改为 `"status": "closed"`，验证系统是否能识别为重复推送，并动态将其关单。

```bash
curl -v -k -X POST https://101.251.208.178/api/ext/tickets \
-H "Content-Type: application/json" \
-H "Authorization: Bearer 1c43f8a0ea1b72b4511d42c584e977e5cea21ba0cce67fcea7a65c079954011d" \
-d '{
  "title": "生产环境联调测试工单 (已关单)",
  "description": "这是一条用于测试联调的工单，验证二次推送自动关单功能。",
  "serviceNo": "TEST-CURL-001",
  "customerName": "VIP大客户",
  "creatorEmployeeId": "TEST_001",
  "assigneeEmployeeId": "TEST_001",
  "type": "software",
  "category1": "远程测试",
  "status": "closed"
}'
```

**预期返回：**
不会报 409 防重错误，而是返回 `"updated": true`，并且 `"status": "closed"`，同时标题自动更新为了最新的带 `(已关单)` 字样的标题。
